# Projects

This course teaches Rust by building projects. Each project introduces a new set of concepts and builds on the previous ones.

| Project | Concepts |
|---------|----------|
| [1 — Caesar Cipher](./caesar/overview.md) | variables, `char`, ASCII, functions, loops, `String`, control flow, `match` |
| [2 — Contact Book](./contact-book/overview.md) | `struct`, `impl`, `Vec<T>`, `Option<T>`, ownership & borrowing |
| [3 — Cipher Cracker](./cipher-cracker/overview.md) | `HashMap`, iterators, closures |
| [4 — A Ball Moves](./ball-moves/overview.md) | `struct`, `impl`, operator overloading, game loop |
| [5 — Many Bodies](./many-bodies/overview.md) | `Vec<T>`, ownership, struct composition |
| [6 — Balls Collide](./collisions/overview.md) | `enum` with data, `Option<T>`, pattern matching |
| [7 — Stable World](./stable-world/overview.md) | `Result<T,E>`, iterators, closures |
| [8 — Boxes](./boxes/overview.md) | traits, generics, SAT |
